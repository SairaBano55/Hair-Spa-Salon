# 💇‍♀️ Hair Spa Salon - Premium Landing Page

A luxurious, modern, and elegant single-page website designed for a premium hair salon based in Karachi. Built from scratch using clean HTML structures and custom CSS styling.

## 🌐 Live Demo
You can view the live website here: [👉 Click Here to View Live Site](https://hair-salon-sb.netlify.app/)

## ✨ Features
- **Premium Dark UI:** Designed with an elegant dark color palette matching luxury salon aesthetics.
- **Interactive Button:** Smooth custom CSS transitions on the "Book An Appointment" call-to-action button.
- **Services & Pricing Menu:** A beautiful translucent table displaying premium services with structured pricing.
- **Clean Typography:** Carefully selected readable modern fonts for a high-end feel.

## 🛠️ Technologies Used
- **HTML5:** For semantic layout, content organization, and tables.
- **CSS3:** For dark overlay effects, background image positioning, transparent panels, and hover interactions.
